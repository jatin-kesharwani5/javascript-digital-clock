# Digital Clock

A simple Digital Clock web application built using HTML, CSS, and Vanilla JavaScript.
This project focuses on working with real-time data and updating the UI dynamically.


## Features

* Displays current hours, minutes, and seconds
* Updates automatically every second
* Proper two-digit time formatting
* Clean and minimal user interface


## Concepts Used

* JavaScript Date object
* DOM Selection (querySelector)
* setInterval for real-time updates
* textContent for updating UI
* Conditional formatting for time values


## How It Works

* The application fetches the current system time using the Date object
* A function runs every second using setInterval
* Hours, minutes, and seconds are updated dynamically on the screen
* Single digit values are formatted into two digits


## How to Run
